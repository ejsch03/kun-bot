# kun-bot
An incredibly based Discord bot.

## Features
- `s.w` responds with a randomly selected image from any provided gallery.
- `s.play/stop/queue...` music functionality.

```
Usage: kun-bot [OPTIONS] <PATHS>... -- <ADMINS>...

Arguments:
  <PATHS>...
  <ADMINS>...

Options:
  -p, --prefix <PREFIX>  [default: s.]
  -t, --title <TITLE>    [default: kun-bot]
  -h, --help             Print help
  -V, --version          Print version
```
| ARG | Description |
| --- | - |
| PATHS | the list of directories containing images |
| ADMINS | the list of user IDs allowing whitelist privileges |
